# 🚀 Pulse — Plataforma de Gestão entre Agência e Cliente

## 📌 Sobre o projeto

O **Pulse** é uma plataforma web desenvolvida para melhorar a comunicação entre **agências de publicidade** e seus **clientes**.

O objetivo principal é resolver problemas comuns do dia a dia, como:

* falta de alinhamento
* retrabalho
* atrasos em entregas
* excesso de mensagens dispersas (WhatsApp, e-mail, etc.)

A ferramenta centraliza tudo em um único lugar, trazendo mais organização, clareza e produtividade.

---

## 🎯 Problema que resolve

Na rotina de agências e clientes, é comum enfrentar:

* ❌ comunicação descentralizada
* ❌ pedidos mal definidos
* ❌ dificuldade no controle de aprovações
* ❌ perda de prazos
* ❌ histórico de decisões inexistente

O Pulse surge para eliminar esses problemas.

---

## 💡 Solução proposta

A plataforma oferece um ambiente centralizado com:

* 📂 gestão de projetos
* ✅ controle de aprovações
* 💬 chat por projeto
* 📝 briefing estruturado
* 🔔 notificações
* 📅 agenda/calendário
* 👥 gestão de clientes

Tudo pensado para tornar a comunicação mais clara e eficiente.

---

## 🧩 Funcionalidades principais

### 📊 Dashboard

* visão geral dos projetos
* status das entregas
* indicadores de progresso

### 📁 Projetos

* acompanhamento de demandas
* controle de prazos
* status (em andamento, concluído, bloqueado)

### ✅ Aprovações

* envio de entregas
* aprovação ou solicitação de alterações
* histórico de decisões

### 💬 Mensagens

* chat por projeto
* comunicação centralizada
* histórico acessível

### 📝 Briefing

* criação de solicitações estruturadas
* definição clara de objetivos
* envio de referências

### 🔔 Notificações

* alertas de prazos
* novas entregas
* atualizações importantes

### 📅 Agenda

* controle de compromissos
* agendamentos com edição e remoção

### 👥 Clientes

* cadastro de clientes
* visualização de perfil
* edição e exclusão
